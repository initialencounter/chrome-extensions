# 验证图片和概要

## 功能介绍

- 通过 `Everything` 工具，搜索图片概要。
- 再调用解析器读取图片概要，与系统检验单进行匹配。

## 前置条件

- 安装解析器。
  - 方法一：安装图形化界面。
  - 方法二：直接运行可执行文件。
- 设置 `Everything` 服务。
- `v1.8.9` 以上版本的 `lims` 插件 。

## 安装

### 方法一：安装图形化界面。

1. 退出所有杀毒软件
2. 双击安装包，进行安装。
3. 选择安装路径，不要安装在C盘。
4. 安装完成后，双击桌面快捷方式 `aircraft` ，运行程序。

### 方法二：可执行文件。

如果方法一程序点击后，没有反应。则使用方法二。

1. 退出所有杀毒软件
2. 解压到C盘以外的地方。
3. 双击 `headless.exe` 运行程序。

### 设置 Everything 服务

1. 打开 `Everything` 选项。
2. 启用 `HTTP服务器` 。
3. 将 `HTTP服务端口` 设置为 `25456` 。
4. 点击 `应用` 按钮。

### 更新 lims 插件

点击验证按钮，验证图片概要。
